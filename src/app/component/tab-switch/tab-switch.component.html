Angular Angular is a frontend framework used to build dynamic, single-page web applications (SPA) using TypeScript.

Node Node.js is a runtime environment that allows you to run JavaScript outside the browser (on the server).

Express Express.js is a lightweight web framework for Node.js that makes it easy to build APIs and web servers.

MongoDB MongoDB is a NoSQL database that stores data in JSON-like format (documents) instead of tables.
